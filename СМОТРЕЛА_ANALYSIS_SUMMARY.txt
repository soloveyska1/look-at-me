6. БЫСТРЫЙ СТАРТ (5 минут)
═══════════════════════════════════════════════════════════════════════════════

ШАГ 1: Откройте ваш DAW (Ableton/FL Studio/Logic Pro/Reaper)

ШАГ 2: Загрузите трек СМОТРЕЛА на мастер-трек

ШАГ 3: Добавьте FabFilter Pro-Q3 (или Waves Q10) на мастер-канал

ШАГ 4: Импортируйте пресет:
        FabFilter: File → Load Preset → СМОТРЕЛА_FabFilter_ProQ3.xml
        Waves:     Load → СМОТРЕЛА_Waves_Q10.json

ШАГ 5: ВКЛЮЧИТЕ LINEAR PHASE для бандов 3, 5, 6 (146, 1693, 2244 Hz)
        В FabFilter: кликните на band → Processing → Linear Phase Medium

ШАГ 6: Постепенно включайте bands:
        а) Сначала bands 3, 5, 6 (критические)
        б) Потом bands 2, 4, 8, 9 (high priority)
        в) Наконец bands 7, 10 (musical boosts)

ШАГ 7: A/B тест - слушайте с EQ и без каждые 30 секунд

ШАГ 8: Компенсируйте output gain (+6 dB после всех cuts)

ШАГ 9: Добавьте Multiband Compressor (настройки в PROFESSIONAL_EQ_GUIDE.txt)

ШАГ 10: Финальный Limiter с ceiling -1.0 dBTP

ГОТОВО! Экспортируйте в WAV 48kHz/24bit

ПЕРЕД ЭКСПОРТОМ ПРОВЕРЬТЕ:

  □ A/B тест с оригиналом выполнен
  □ Проверено на студийных мониторах
  □ Проверено в наушниках
  □ Проверено на смартфоне/ноутбуке (consumer playback)
  □ Сравнено с 2-3 референсными треками в жанре
  □ LINEAR PHASE включен для критических частот
  □ Output gain компенсирован
  □ Multiband compression применен
  □ Final limiter настроен (-1.0 dBTP)
  □ LUFS проверен (анализатор loudness)
  □ True Peak не превышает -1.0 dBTP
  □ Нет фазовых проблем (моно совместимость)
  □ Listening fatigue test пройден (1+ час прослушивания)

9. PRO TIPS
═══════════════════════════════════════════════════════════════════════════════

✓ ДЕЛАЙТЕ:
  • Применяйте EQ cuts постепенно - не все сразу!
  • Делайте перерывы каждые 30-45 минут
  • Слушайте на разной громкости (тихо/средне/громко)
  • Доверяйте своим ушам больше, чем визуальным анализаторам
  • Сравнивайте с коммерческими референсами

✗ НЕ ДЕЛАЙТЕ:
  • Не booстите больше +3-4 dB за раз
  • Не микшируйте только в наушниках
  • Не забывайте про gain staging
  • Не применяйте чужие пресеты без адаптации
  • Не пропускайте A/B тестирование
