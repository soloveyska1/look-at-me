## Кастомизация

### Изменить цвета

Открыть `render_teaser_v3_branded.py`, найти:

```python
# Brand colors (строка ~45)
COLOR_BACKGROUND = (12, 12, 12)      # Черный
COLOR_TEXT_PRIMARY = (252, 252, 250) # Белый
COLOR_ACCENT_GREEN = (48, 168, 107)  # Зеленый (ГЛАВНОЕ!)
```

### Изменить тайминг

Открыть `render_teaser_v3_branded.py`, найти:

```python
# Intro card timing (строка ~220)
if t < 0.5:      # Pure black
if t < 1.5:      # Eye symbol
if t < 2.0:      # Fade in text
elif t < 3.5:    # Hold
else:            # Fade out
```

### Изменить шрифты

```python
# В функции create_intro_card_saymurr() и create_outro_card_full()
main_font = ImageFont.truetype(
    "/usr/share/fonts/truetype/dejavu/DejaVuSans-Bold.ttf",
    140  # Размер
)
```
